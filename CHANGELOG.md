# Changelog

## [0.1.0] - 2026-05-13

### Added
- Initial release
- Webview UI with step-by-step generation workflow
- AI-powered data generation via GitHub Copilot LLM
- Faker.js fallback for offline/no-Copilot scenarios
- Multi-table generation with topological dependency sorting
- Automatic relationship resolution via `@odata.bind`
- Business context input for domain-specific data
- Column filtering: Best Applicable, Mandatory Only, Selected Columns
- Batch writes using Dataverse `$batch` API
- Cleanup mode with reverse topological sort deletion
- Sort order filtering (newest/oldest first)
- FetchXML filter support for targeted deletion
- Deletion plan preview with confirmation
- Browser Sign-in (PKCE) and Device Code authentication
- GitHub Copilot Chat participant (`@dvdata`)
- Language Model Tool for programmatic invocation
- Selected table/column chips for quick visibility
- Collapsible table picker sections
- Reset button for clearing all selections
